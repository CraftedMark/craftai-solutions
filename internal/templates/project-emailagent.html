Usage Examples

Simple commands to get started with powerful automation

Single Account Processing

# Process both receipts and spam
python3 email_agent.py

# Process only receipts
python3 email_agent.py --receipts-only

# Custom export path
python3 email_agent.py --export-path /path/to/exports

Multi-Account Processing

# Process all accounts in parallel
python3 multi_email_agent.py --parallel

# Process specific account only
python3 multi_email_agent.py --account "Personal Gmail"

# Receipts only for all accounts
python3 multi_email_agent.py --receipts-only
